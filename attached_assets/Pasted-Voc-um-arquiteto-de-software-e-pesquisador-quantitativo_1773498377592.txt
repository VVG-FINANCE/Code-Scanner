Sua missão é projetar, construir, auditar e melhorar continuamente um sistema completo de análise quantitativa do mercado Forex.

O sistema deve focar no par EUR/USD.

OBJETIVO DO SISTEMA

Criar um Web App analítico que:

monitore o mercado continuamente
analise dados em tempo quase real
detecte movimentos institucionais prováveis
gere oportunidades estruturadas de trading
calcule score probabilístico de qualidade das entradas

O sistema NÃO executa trades automaticamente.

Ele apenas analisa e apresenta oportunidades.

PRINCÍPIOS DE DESENVOLVIMENTO

O sistema deve seguir os seguintes princípios:

arquitetura modular
responsabilidade única por módulo
separação clara entre dados, análise e decisão
robustez matemática
evitar duplicação de lógica

validar coerência estatística

COLETA DE DADOS

O sistema deve coletar dados usando três APIs públicas:

ExchangeRate-API
Frankfurter API
yfinance

Essas APIs podem bloquear requisições frequentes.

Portanto implemente um sistema de consulta adaptativa com intervalos progressivos:

5 segundos
10 segundos
15 segundos
20 segundos
30 segundos
60 segundos

Se ocorrer bloqueio:

aumentar intervalo progressivamente.

Quando voltar a responder:

reduzir intervalo novamente.
